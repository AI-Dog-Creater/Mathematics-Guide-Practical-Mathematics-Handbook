# 📖 Introduction

Mathematics is the foundation of science and engineering. This project aims to build a **systematic, comprehensive, and open mathematical knowledge repository**, collecting essential theories, formulas, methods, and classical results across major mathematical disciplines.

This guide covers both **pure mathematics and applied mathematics**, providing a structured reference for students, researchers, engineers, and anyone interested in mathematics.

The repository includes:

- Analysis
- Algebra
- Geometry
- Foundations of Mathematics
- Calculus of Variations and Optimization
- Probability Theory and Mathematical Statistics
- Computational Mathematics and Scientific Computing
- History of Mathematics

The content contains a large collection of mathematical tools and theories, including:

- Infinite Series
- Special Functions
- Differential and Integral Calculus
- Multiple Integrals
- Integral Transforms
- Ordinary and Partial Differential Equations
- Mathematical Statistics
- Numerical Analysis
- Scientific Computing Methods

The goal of this project is to provide a long-term mathematical reference that connects fundamental theories with modern scientific applications.

# 📖 项目简介

数学是科学与工程领域的基础。

本项目旨在构建一个**系统、全面、开放的数学知识库**，整理数学领域中的重要理论、公式、方法以及经典结果。

本指南覆盖纯数学、应用数学以及计算数学的重要方向，为学生、科研人员、工程师以及数学爱好者提供长期参考。

主要内容包括：

- 分析学
- 代数学
- 几何学
- 数学基础
- 变分法与优化
- 概率论与数理统计
- 计算数学与科学计算
- 数学史

同时包含大量数学工具与理论：

- 无穷级数
- 特殊函数
- 微积分理论
- 多重积分
- 积分变换
- 常微分方程与偏微分方程
- 数理统计方法
- 数值分析
- 科学计算方法

本项目希望建立一个连接基础数学理论与现代科学应用的开放数学资源平台。
